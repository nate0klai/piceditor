Скромный редактор списка фотографий
***
Для запуска проекта необходимо выполнить:
```
npm i
npm start
```
***
Назначение папок |
-----------------|----------------------
src/             |   # Исходные файлы.
  actions/       |   # - actions.
  components/    |   # - "глупые" компоненты.
  containers/    |   # - "умные" компоненты.
  reducers/      |   # - reducers.
  store/         |   # - файлы хранилища.
  styles/        |   # - файлы стилей.
